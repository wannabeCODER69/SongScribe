# SongScribe

AI-powered subtitle generator built with Node.js, Express, FFmpeg, Whisper, and React.

## Current Features

- Upload audio/video
- Organize uploads into job folders
- Extract audio using FFmpeg

## Planned Features

- Whisper transcription
- Subtitle generation (SRT)
- Subtitle editor
- React frontend
- Subtitle export
