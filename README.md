# SongScribe

An intelligent lyrics generation and synchronization tool.

SongScribe automatically identifies songs and retrieves synchronized lyrics from online databases when available. If lyrics cannot be found, it falls back to local AI transcription using Faster-Whisper.

## Features

- 🎵 Song identification
- 📚 LRCLIB integration
- 🤖 AI fallback transcription
- 🎤 Audio extraction with FFmpeg
- ✏️ Editable lyrics
- 📄 Export to LRC, SRT and TXT
- 💻 Open source
- 🌐 Cross-platform
